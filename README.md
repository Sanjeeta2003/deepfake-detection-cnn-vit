# deepfake-detection-cnn-vit
Deepfake Detection using CNN and Vision Transformer

## 📌 Overview
This project detects deepfake images/videos using a hybrid model combining 
Convolutional Neural Networks (CNN) and Vision Transformers (ViT).

## 🧠 Model Architecture
- CNN: Feature extraction from local regions
- Vision Transformer (ViT): Global attention-based classification

## 📂 Files
- `deepfake_detection.ipynb` — Main notebook

## 🚀 How to Run
1. Open the notebook in [Google Colab](https://colab.research.google.com/)
2. Install dependencies: `pip install -r requirements.txt`
3. Run all cells

## 📊 Dataset
Faceforensics++ c-23
